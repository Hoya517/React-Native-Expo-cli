#expo-cli
## Install
```
npm install -g expo-cli // -g 옵션은 global 옵션
// 위의 명령어가 안되면 
// npm install --legacy-peer-deps --global expo-cli
```
```
expo --version  // 4.5.0
```

## Project
```
expo init [ProjectName]  // blank 선택(TypeScript는 익숙X)
```
```
expo start  // http://localhost:19002
```
```
Press w  // http://localhost:19006
```
